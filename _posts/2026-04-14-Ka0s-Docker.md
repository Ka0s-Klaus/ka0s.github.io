Docker no es “una dependencia”: es parte del plano de ejecución.

El módulo `core/docs/ka0s_docker/01_concept.md` se centra en 3 objetivos:

- estandarizar versión de Docker Engine en los nodos,
- habilitar operación visual (Portainer),
- asegurar configuración base (redes/volúmenes).

Cuando Docker está estandarizado, todo lo demás se vuelve más reproducible.
