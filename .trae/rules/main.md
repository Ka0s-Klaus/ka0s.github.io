# Ka0s Web Constitution (Web Governance)

Este archivo define las reglas simplificadas para los repositorios web (`ka0s.github.io`, `www.ka0s.io`).

## 📜 Reglas Aplicables

0.  **[Regla 000: Identidad](rules_library/rule_000_identidad.md)**
    *   Identidad Ka0s y excelencia.
1.  **[Regla 001: Verificación](rules_library/rule_001_verificacion.md)**
    *   Prohibido commit directo a main sin validación local (build/lint).
2.  **[Regla 003: GitOps](rules_library/rule_003_gitops.md)**
    *   El repositorio es la fuente de verdad.
3.  **[Regla 010: Documentación](rules_library/rule_010_documentation.md)**
    *   Estructura clara y mantenimiento de `README.md`.
